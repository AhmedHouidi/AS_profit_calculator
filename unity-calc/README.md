# Unity Publisher Calculator

A mobile-friendly web app for Unity Asset Store publishers to calculate profit, compare discount scenarios, and track payouts.

## Features
- **Profit calculator** — enter price, sales, refund rate & discount → see exact earnings
- **Discount compare** — compare all 4 Unity discount tiers side by side
- **Payout tracker** — track progress toward PayPal ($100) or Wire transfer ($250) threshold

## Deploy to GitHub Pages
1. Push this repo to GitHub
2. Go to **Settings → Pages**
3. Set source to **Branch: main**, folder **/ (root)**
4. Save — your app will be live at `https://yourusername.github.io/repo-name`

## Install on Android
Once deployed, open the URL in Chrome on Android → tap the **"Add to Home Screen"** banner → installed as an app.

## Tech
Plain HTML + CSS + JS. No dependencies, no build step. Works fully offline after first visit.
